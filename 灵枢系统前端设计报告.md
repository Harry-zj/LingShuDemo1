# 灵枢系统前端设计报告

> **版本**：v2.0 | **日期**：2026年7月 | **设计主题**：LingShu × Tool of NA — 黑底胶片感视觉体系

---

## 一、设计理念与视觉定位

灵枢系统采用 **"Tool of NA" 暗黑胶片美学**，以深邃黑底为画布，金色（#f4b847）为主色调、青色（#00d4ff）为辅助点缀，营造高对比度、强视觉冲击力的智能教务平台质感。

**核心设计关键词**：
- **黑底胶片感** — 深黑基底（#050505），半透明玻璃态面板，高光对比
- **超大标题** — clamp 动态字号，h1 可达 112px，强调视觉层级
- **圆角影像卡** — 大圆角（24-36px），深阴影，悬浮上浮交互
- **经络流动** — 金色/青色光晕动画，模拟 AI 数据流转
- **五维配色** — 智育蓝、德育金、体育绿、美育粉、劳育橙

### 7.3 个性化评定报告（ReportView.vue）

**布局**：65/35 双栏，右侧 sticky

**左 65%**：学生档案 → AI 综合总评（DeepSeek 生成，可重新生成）→ 分阶段提升方案（双栏：本月/期末前 + 学年内，勾选框交互）→ 成长目标（左侧预览 + 右侧记事本编辑器）→ 历年回顾（文字时间轴，按批次过滤）

**右 35%**：五维个性画像（5 张维度卡片，含分数/等级/优弱标签，点击跳转活动指南）→ 本学年亮点 → 待提升事项 → 操作功能区（导出 PDF / 打印 / 保存规划 / 分享报告 / 生成年度总结）

### 7.4 维度活动指南（DimensionDetail.vue）

**布局**：左侧 50% 图片轮播区 + 右侧活动卡片

**图片轮播**：有图自动播放 3.5s + 左右箭头 + 底部圆点。无图显示虚线占位框。

**活动卡片**：序号圆圈 + 标题 + 分类标签 + 描述 + 官网链接 + 添加至目标按钮。

**智维度**支持专业匹配优先推荐。

**五维度活动池**：德 16 项 / 智 30+ 项 / 体 16 项 / 美 19 项 / 劳 5 项。

### 7.5 智能填表（zongce/SmartFill.vue）

四步工作流：
1. **规则管理**（SmartFillRule）— 上传评分规则文件或输入文本，AI 解析为结构化规则
2. **材料识别**（SmartFillMaterial）— 创建材料条目，上传附件，AI 分析给出分类/分数/置信度
3. **评分清单**（SmartFillScore）— 总分 + 五维分项得分明细
4. **自动填表**（SmartFillForm）— 上传 .docx 模板，数据预览，一键填充下载

## 十二、数据流

```
用户交互 → Vue Component → api/模块 API → /api/* 代理
                                              ↓
                                        后端响应 JSON
                                              ↓
                           ┌──────────────────┴──────────────────┐
                           │                                      │
                    即时数据（分数/排名）                  AI 内容（评语/方案）
                    → 直接渲染                                → MySQL report_cache
                                                              → 跨设备同步
                           │                                      │
                           └──────────────┬───────────────────────┘
                                          ↓
                                    组件状态更新
                                    (reactive/computed)
                                          ↓
                                    DOM 自动渲染
```

**客户端缓存**：
- 维度活动选择 → `localStorage`（用户 + 维度 + 版本号）
- 报告 AI 缓存 → 服务端 `report_cache` 表
